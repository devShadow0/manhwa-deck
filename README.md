# Manhwa Deck 📚✨

A premium, modern desktop application designed for discovering, reading, and managing your favorite manhwa, manga, and manhua in one place. Search across supported websites, read chapters in a custom-designed ad-free reader, download chapters for offline access, and track your reading history seamlessly.

---

## ✨ Features

- 🎭 **Custom Reader Overlay:** Custom-designed, distraction-free reading deck with controls for page zoom, brightness adjustment, and background dimming.
- 📥 **Offline Downloader:** Download complete chapters to your local storage to read offline anytime.
- ⚡ **Smart Prefetching:** Pre-loads upcoming pages and chapters in the background as you read, eliminating loading times between chapters.
- 🚫 **Built-in Ad-Blocker:** Preconfigured ad-blocking engine rules targeting webview loads so reading remains completely ad-free.
- 🔄 **Automatic Updates:** Checks for new chapter releases and application updates seamlessly on startup.
- 🎨 **Harmony Aesthetics:** Premium dark mode UI built with custom gradients, interactive glassmorphic panels, and smooth micro-animations.
- 📂 **Library & History Management:** Pin favorites, organize titles, track progress, and resume reading where you left off.
